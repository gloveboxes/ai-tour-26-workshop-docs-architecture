## Mga Mapagkukunan

Ang seksyong ito ay dapat magbigay ng mga kapaki-pakinabang na mapagkukunan para sa mga kalahok ng workshop upang ipagpatuloy ang kanilang pag-aaral at mas malalim na maunawaan ang mga teknolohiyang saklaw.

## Halimbawa ng Mga Kategorya ng Mapagkukunan

### Dokumentasyon at Gabay
- Mga opisyal na link ng dokumentasyon
- Mga gabay sa pagsisimula
- Dokumentasyon ng pinakamahusay na kasanayan
- Mga API reference

### Mga Materyales sa Pag-aaral
- Mga online na kurso at tutorial
- Mga video demonstration
- Mga sample na repository ng code
- Mga forum ng komunidad at talakayan

### Mga Tool at Utility
- Mga development tool at extension
- Mga testing framework
- Mga monitoring at debugging tool
- Mga deployment utility

### Komunidad at Suporta
- Mga opisyal na channel ng komunidad
- Mga Stack Overflow tag
- Mga GitHub repository
- Mga blog at artikulo ng developer

## Halimbawa ng Format ng Entry

**Pangalan ng Tool/Mapagkukunan**
- Maikling paglalarawan ng kung ano ang ibinibigay nito
- Link sa mapagkukunan
- Bakit ito mahalaga sa nilalaman ng workshop
