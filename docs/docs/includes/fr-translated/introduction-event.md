## Commencer - Participants à l'Événement

Bienvenue à l'atelier ! Suivez ces étapes simples pour commencer.

## Prérequis

- Vous devriez avoir reçu un lien vers un environnement de laboratoire Skillable
- Assurez-vous d'avoir une connexion Internet stable
- Ayez vos identifiants d'événement prêts

## Étapes de démarrage rapide

### 1. Accédez à votre environnement de laboratoire
- Cliquez sur le lien du laboratoire Skillable fourni par les organisateurs de l'événement
- Connectez-vous en utilisant vos identifiants d'événement
- Attendez que votre environnement de laboratoire personnalisé se charge (cela peut prendre 2-3 minutes)

### 2. Vérifiez votre environnement
Une fois votre laboratoire chargé, vous devriez voir :
- Une interface VS Code dans votre navigateur
- Les fichiers de l'atelier déjà chargés dans l'espace de travail
- Un terminal prêt à l'emploi

### 3. Testez votre configuration
- Ouvrez le terminal dans VS Code
- Exécutez une commande simple pour vérifier que tout fonctionne :
  ```bash
  echo "Environnement d'atelier prêt !"
  ```

### 4. Vous êtes prêt !
Votre environnement est maintenant configuré et prêt pour l'atelier. L'instructeur vous guidera à travers les prochaines étapes.

## Besoin d'aide ?

- Levez la main pour obtenir de l'aide des facilitateurs de l'atelier
- Consultez votre voisin - il pourrait avoir rencontré le même problème
- Recherchez la section "Aide" dans votre environnement Skillable

**Important :** Gardez votre onglet de navigateur ouvert tout au long de l'atelier pour maintenir votre session.
