Terminer - Apprenants Autonomes¶

Félicitations pour avoir terminé l'atelier ! Voici comment nettoyer correctement votre environnement et poursuivre votre parcours d'apprentissage.

Nettoyez votre environnement¶

1. Sauvegardez vos progrès¶

  • Validez et poussez vos modifications dans votre dépôt forké : 
    git add .
git commit -m "Exercices de l'atelier terminés"
git push origin main
  • Votre travail est maintenant sauvegardé en toute sécurité dans votre dépôt GitHub

2. Arrêtez votre Codespace GitHub¶

  • Allez sur github.com/codespaces
  • Trouvez votre Codespace d'atelier
  • Cliquez sur les trois points (...) et sélectionnez "Arrêter le codespace"
  • Cela évite des frais d'utilisation inutiles

3. Nettoyage des ressources¶

  • Supprimez tous les fichiers de modèles volumineux que vous ne prévoyez pas d'utiliser : 
    # Exemple : Supprimer les modèles téléchargés
rm -rf ./models/large-model-files
  • Effacez les fichiers temporaires et les caches si nécessaire
  • Conservez uniquement le code et les configurations que vous souhaitez consulter plus tard

4. Surveillez votre utilisation¶

  • Vérifiez votre utilisation de GitHub Codespaces dans Paramètres > Facturation
  • Les comptes gratuits ont des limites mensuelles - soyez attentif à vos heures restantes
  • Envisagez une mise à niveau si vous prévoyez de faire plus de développement

Poursuivez votre apprentissage¶

Prochaines étapes¶

  • Explorez le dépôt de l'atelier pour des exemples supplémentaires
  • Essayez de mettre en œuvre les concepts dans un nouveau projet
  • Expérimentez avec différentes configurations et paramètres

Conseils de gestion des ressources¶

  • Utilisez Codespaces uniquement pour le développement actif
  • Arrêtez Codespaces pendant les pauses
  • Envisagez de configurer un environnement de développement local pour un apprentissage prolongé

Restez connecté¶

  • Ajoutez une étoile au dépôt de l'atelier - Cliquez sur le bouton étoile sur GitHub pour le mettre en signet pour un accès facile plus tard
  • Participez aux discussions et forums communautaires
  • Suivez les créateurs de l'atelier pour des mises à jour et du nouveau contenu

Rappel : Vous pouvez toujours redémarrer votre Codespace plus tard pour continuer à expérimenter avec les matériaux de l'atelier !
