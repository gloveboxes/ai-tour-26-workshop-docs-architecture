Commencer - Apprenants Autonomes¶

Bienvenue à l'atelier en auto-apprentissage ! Suivez ces étapes pour configurer votre environnement de développement à l'aide de GitHub Codespaces.

Prérequis¶

  • Un compte GitHub (le niveau gratuit est suffisant)
  • Un navigateur Web (Chrome, Firefox, Safari ou Edge)
  • Une familiarité de base avec VS Code (utile mais pas obligatoire)

Étapes de démarrage rapide¶

1. Forkez le dépôt¶

  • Accédez au dépôt de l'atelier sur GitHub
  • Cliquez sur le bouton "Fork" pour créer votre propre copie
  • Cela garantit que vous pouvez apporter des modifications et enregistrer vos progrès

2. Lancez GitHub Codespaces¶

  • Dans votre dépôt forké, cliquez sur le bouton vert "Code"
  • Sélectionnez l'onglet "Codespaces"
  • Cliquez sur "Créer un codespace sur main"
  • Attendez que votre environnement se construise (généralement 2 à 5 minutes)

3. Vérifiez votre environnement¶

Une fois que Codespaces se charge, vous devriez voir : - Une interface VS Code s'exécutant dans votre navigateur - Les fichiers de l'atelier dans le panneau Explorateur - Un terminal en bas de l'écran

4. Testez votre configuration¶

  • Ouvrez le terminal intégré
  • Exécutez cette commande pour vérifier que tout fonctionne : 
    echo "Environnement d'atelier en auto-apprentissage prêt !"

5. Vous êtes prêt à apprendre !¶

Votre environnement de développement est maintenant configuré. Vous pouvez travailler à votre rythme à travers l'atelier.

Conseils pour l'auto-apprentissage¶

  • Prenez votre temps avec chaque section
  • Expérimentez avec les exemples de code
  • Utilisez le terminal pour explorer et tester des commandes
  • Votre Codespace enregistrera automatiquement vos progrès

Besoin d'aide ?¶

  • Consultez la documentation de l'atelier pour des explications détaillées
  • Examinez attentivement les messages d'erreur - ils contiennent souvent des indices utiles
  • Utilisez les Issues GitHub pour poser des questions ou signaler des problèmes

Remarque : Codespaces a des limites d'utilisation sur les comptes gratuits. Surveillez votre utilisation dans vos paramètres GitHub.
